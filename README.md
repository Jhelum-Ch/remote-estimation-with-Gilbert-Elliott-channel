# remote-estimation-with-Gilbert-Elliott-channel
This repository contains codes for remote estimation problem with two agents connected via Gilbert-Elliott channel (erasure channel with Markovian packet drops). The code is for the example given in 'Remote estimation over a packet-drop channel with Markovian state' (https://www.jhelumch.com/wp-content/uploads/2018/09/JC-AM-markov_RE.pdf)
